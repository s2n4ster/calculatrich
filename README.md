# Modern Calculator

Современный калькулятор в стиле iPhone с тремя вкладками: Стандартный, Инженерный и Программист.

## Установка и запуск

```bash
git clone https://github.com/s2n4ster/calculatrich.git
cd calculatrich
pip install -r requirements.txt
python calculator.py
```

## Функции

### Стандартный режим
- Базовые операции (+, -, ×, ÷)
- Процентные вычисления
- Смена знака (±)
- Очистка (C)

### Инженерный режим
- Тригонометрические функции (sin, cos, tan)
- Логарифмы (log, ln)
- Степени (x², x³, x^y)
- Корень квадратный (√)
- Константы (π, e)
- Факториал (n!)
- Обратное число (1/x)

### Режим программиста
- Системы счисления (HEX, DEC, OCT, BIN)
- Логические операции (AND, OR, XOR, NOT)
- Битовые сдвиги (LSH, RSH, ROL, ROR)
- Шестнадцатеричные символы (A-F)

## Особенности

- Современный дизайн в стиле iPhone
- Скругленные кнопки и элементы
- Темная тема
- История вычислений
- Поддержка клавиатуры
- Адаптивный интерфейс с вкладками
- Использует CustomTkinter для современного UI
